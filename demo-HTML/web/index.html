Guía de HTML - Fundamentos

Los elementos esenciales para empezar a crear páginas web

1Estructura básica de HTML

  • Todo documento HTML comienza con <!DOCTYPE html> (declara HTML5)
  • <html>: elemento raíz que contiene todo el documento
  • <head>: metadatos, título, enlaces a CSS/JS (no visible)
  • <body>: contenido visible de la página
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi página</title>
</head>
<body>
    <!-- Contenido aquí -->
</body>
</html>

2Metadatos (head)

  • <meta charset="UTF-8">: codificación de caracteres
  • <meta name="viewport">: responsive en móviles
  • <title>: título en la pestaña del navegador
  • <link>: enlaza CSS externos
  • <script>: enlaza o incluye JavaScript
  • <meta name="description">: SEO
<link rel="stylesheet" href="estilos.css">
<script src="app.js"></script>

3Encabezados

  • Definen la jerarquía del contenido
  • 6 niveles: <h1> a <h6>
  • Solo un <h1> por página (SEO)
  • No saltar niveles (h1 → h2 → h3...)

H1 - Título principal

H2 - Secundario

H3 - Terciario

H4 - Cuarto nivel

H5 - Quinto nivel
H6 - Sexto nivel

4Párrafos y texto

  • <p>: párrafo de texto
  • <br>: salto de línea
  • <hr>: línea horizontal separadora
  • <strong>: texto importante (negrita)
  • <em>: énfasis (cursiva)
  • <mark>: texto resaltado
  • <small>: texto pequeño

Este es un párrafo con texto importante y con énfasis.

5Enlaces

  • <a href="url">: crea hipervínculos
  • href: URL destino (relativa o absoluta)
  • target="_blank": abre en nueva pestaña
  • href="#id": ancla a elemento en la misma página
  • href="mailto:email": abre cliente de correo
  • href="tel:numero": llama en móviles

6Imágenes

  • <img>: elemento vacío (sin cierre)
  • src: ruta de la imagen (obligatorio)
  • alt: texto alternativo (accesibilidad, obligatorio)
  • width / height: dimensiones
  • loading="lazy": carga diferida (rendimiento)
  • Formatos: JPG, PNG, GIF, SVG, WebP
<img src="foto.jpg" alt="Descripción" width="200">

7Listas

  • <ul>: lista desordenada (viñetas)
  • <ol>: lista ordenada (números)
  • <li>: elemento de lista
  • <dl>, <dt>, <dd>: lista de definiciones
  • Las listas pueden anidarse

8Contenedores: Div y Span

  • <div>: contenedor de bloque (agrupa secciones)
  • <span>: contenedor en línea (dentro de texto)
  • No tienen significado semántico
  • Útiles para aplicar CSS y JavaScript
Este es un <div> con un span resaltado dentro.

9Tablas

  • <table>: contenedor de la tabla
  • <thead>, <tbody>, <tfoot>: secciones
  • <tr>: fila (table row)
  • <th>: celda de encabezado
  • <td>: celda de datos
  • colspan / rowspan: combinar celdas

10Formularios

  • <form>: contenedor del formulario
  • <input>: campo de entrada (text, email, password, number, etc.)
  • <label>: etiqueta asociada al input (accesibilidad)
  • <textarea>: área de texto multilínea
  • <select> + <option>: lista desplegable
  • <button>: botón de acción

11Tipos de Input

  • text, password, email, number
  • tel, url, date, time
  • checkbox, radio
  • file, color, range
  • Atributos: required, placeholder, disabled

12Elementos semánticos

  • <header>: encabezado de página/sección
  • <nav>: navegación principal
  • <main>: contenido principal (único)
  • <article>: contenido independiente
  • <section>: sección temática
  • <aside>: contenido relacionado (sidebar)
  • <footer>: pie de página/sección
<header>Logo y navegación</header>
<main>
  <article>Contenido principal</article>
  <aside>Sidebar</aside>
</main>
<footer>Copyright</footer>

13Multimedia

  • <video>: insertar vídeo
  • <audio>: insertar audio
  • <source>: múltiples formatos
  • Atributos: controls, autoplay, loop, muted
  • <iframe>: incrustar contenido externo (YouTube, mapas)
<video controls width="300">
  <source src="video.mp4" type="video/mp4">
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

<iframe src="https://youtube.com/embed/ID"></iframe>

14Atributos globales

  • id: identificador único
  • class: clase(s) CSS
  • style: estilos en línea
  • title: tooltip al pasar el mouse
  • data-*: datos personalizados
  • hidden: oculta el elemento
  • lang: idioma del contenido

Pasa el mouse aquí

<div id="unico" class="caja grande" data-info="valor">

15Buenas prácticas

  • Usar siempre <!DOCTYPE html>
  • Declarar lang en <html>
  • Incluir charset y viewport
  • Usar elementos semánticos
  • Siempre incluir alt en imágenes
  • Asociar <label> a cada input
  • Validar HTML (validator.w3.org)
  • Indentar correctamente el código
  • Separar estructura (HTML), estilo (CSS) y comportamiento (JS)
  • Mantener el código limpio y comentado
